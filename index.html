A inteligência por trás da cor acessível.

O que é o Croma?

O Croma não é apenas uma interface; é uma biblioteca de lógica. Ele foi criado para resolver a subjetividade da cor, oferecendo parâmetros físicos, biológicos e técnicos imutáveis para quem desenvolve ferramentas de acessibilidade.

Para Desenvolvedores

Integre o CromaCore.js no seu projeto para gerar metadados de cor automaticamente.

  • Cálculo dinâmico de CUdC.
  • Sem dependências externas.
  • Ficha técnica multissensorial.

Para Pesquisadores

Utilize a base fundamentada em Vigotski para criar novas metodologias de ensino inclusivo.

  • Física da cor (Termodinâmica).
  • Psicologia Evolutiva.
  • Âncoras Táteis Reais.

Exemplo de Integração

// Inicializando o motor
const croma = new CromaCore();

// Consultando o "Verde Claro"
const info = croma.gerarFicha('amarelo', 'azul', 'claro');

console.log(info.tecnico.codigo_string); 
// Resultado: ç + 13

O que é o Croma?

What is Croma?

O Croma não é apenas uma interface; é uma biblioteca de lógica. Ele foi criado para resolver a subjetividade da cor, oferecendo parâmetros físicos, biológicos e técnicos imutáveis para quem desenvolve ferramentas de acessibilidade.

Croma is not just an interface; it is a logic library. It was created to solve color subjectivity, offering immutable physical, biological, and technical parameters for those developing accessibility tools.
